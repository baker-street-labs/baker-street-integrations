Future plans and development priorities for Baker Street Integrations.

## Planned Improvements

### Enhanced Certificate Automation

**Status**: Planned

- Automated certificate renewal monitoring
- Certificate expiration alerting
- Automated renewal workflows
- Certificate revocation automation
- Multi-domain support

### Advanced PANOS Integration

**Status**: Planned

- Policy rule automation
- Security profile management
- Dynamic address group updates
- Threat intelligence feeds (EDLs)
- Panorama device group management

### Torq Workflow Enhancements

**Status**: Planned

- Additional workflow templates
- Advanced error handling and retry logic
- Workflow versioning and rollback
- Performance optimization
- Integration with additional tools

### Multi-Vendor Support

**Status**: Planned

- Additional firewall vendors (Fortinet, Check Point)
- SIEM integrations (Splunk, QRadar)
- Cloud provider integrations (AWS, Azure, GCP)
- Container orchestration platforms (Kubernetes, OpenShift)
